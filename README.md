In order to make use of these files you need to create a terraform.tfvars and specify the following variables:

vcd_user
vcd_pass 
vcd_org
vcd_vdc
vcd_url = "https://some-vcloud.com/api"
vcd_max_retry_timeout
vcd_allow_unverified_ssl
vcd_catalog_name
vcd_catalog_id
vapp_template_name
vapp_template_id
network_name
